# Setup - Documentação de Configuração

Esta pasta contém toda a documentação relacionada à configuração inicial do projeto.

## Arquivos

- **SETUP.md** - Guia completo de configuração AWS usando OIDC com Roles
- **VERIFY_SETUP.md** - Como verificar se a configuração está correta
- **NEXT_STEPS.md** - Próximos passos após a configuração inicial
- **UPDATE_ROLE.md** - Como atualizar ou recriar a IAM Role
- **VERCEL_SETUP.md** - Configuração do deploy do frontend na Vercel

## Ordem Recomendada de Leitura

1. Comece com **SETUP.md** para configurar o ambiente AWS
2. Use **VERIFY_SETUP.md** para confirmar que tudo está funcionando
3. Consulte **VERCEL_SETUP.md** para configurar o deploy do frontend
4. Consulte **NEXT_STEPS.md** para saber o que fazer depois
5. Se precisar atualizar a Role, veja **UPDATE_ROLE.md**
